Ferramentas Utilizadas:

IDE - Netbeans v8.2
Selenium Java - v3.141.59
SO - Windows 10

Instruçoes:
	
	No NetBeans: 
		- Criar novo projeto Maven
		- Adicionar nova dependencia
		- Preencher o formulario da dependencia com as info da versao pretendida, podem encontrá-las em 
		https://mvnrepository.com/artifact/org.seleniumhq.selenium/selenium-java 

	Localmente:
		- Visitar e fazer download do Driver adequado ao sistema usado no caso do Google Chrome em, 
		https://sites.google.com/a/chromium.org/chromedriver/downloads, e no caso do Firefox, em https://github.com/mozilla/geckodriver.
		- Descomprimir o driver e colocá-lo no path C:/Windows/System32
